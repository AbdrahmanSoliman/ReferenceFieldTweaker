## Requirements
- Unity 2020.3+

## Installation
### Unity Package Manager
1. Open the Package Manager (Window > Package Manager)
2. Click the '+' button > "Add package from git URL"
3. Enter: `https://github.com/AbdrahmanSoliman/ReferenceFieldTweaker.git`
